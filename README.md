## Setup

Install Python libraries via Poetry by the following commands:

```sh
pip install poetry
poetry install -E $EXTRAS
poetry shell
```

You must choose `$EXTRAS` from the following options:

- `fastscnn`: Fast-SCNN dependencies
- `vis`: Visualization dependencies
